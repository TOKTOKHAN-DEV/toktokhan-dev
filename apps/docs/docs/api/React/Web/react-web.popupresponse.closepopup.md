팝업을 닫는 함수를 나타냅니다. 팝업이 닫힐때 부모 창에 OAuth 응답 데이터를 전달합니다. `useOauthPopupListener` 훅을 사용하여 부모 창에서 OAuth 응답 데이터를 수신할 수 있습니다.

## Signature

```typescript
closePopup(extra?: any): void;
```
