네이버 소셜 로그인 아이콘 버튼 UI 컴포넌트입니다. 아이콘 버튼으로 사용되며 레이블과 함께 표기됩니다. [`IconButton`](./react-web.iconbutton) 컴포넌트를 기반으로 하며, 네이버 아이콘 및 버튼 스타일링이 가능합니다.
