똑똑한개발자의 플러그인을 사용할 수 있는 **CLI TOOL**입니다. `gen:api`, `gen:theme`, `gen:route`, `gen:sitemap` 등 다양한 플러그인을 지원하며, 사용자가 개발한 스크립트를 플러그인으로 등록할 수 있습니다.
