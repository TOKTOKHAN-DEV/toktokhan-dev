# Introduction

## New Tokript With Plugin

새로워진 모습으로 Tokript 가 돌아왔습니다.<br/>
기존과 다르게 각 기능들을 모듈화 함으로써 더욱 유연하게 스크립트를 사용할 수 있게 되었습니다.

#### 기존에는 어떤 문제가 있었나요?

기존 tokript 는 하나의 보일러 템플릿을 맞춤으로 제작된 맞춤형 cli package 였습니다.

1. 프로젝트마다 많은 변수를 고려하기위해 configuration 을 통해서, 여러 인풋을 받게 만들 수 밖에 없었습니다.
2. 특정 프로젝트에선 단 하나의 기능을 사용하기 위해 사용하지 않는 패키지를 모두 설치해야 했습니다.
3. 사용 환경이 제한적이었습니다. 만약 react native 에서 사용하고 싶다면, 기존 tokript 를 사용할 수 없었고, rn-tokript 와 같은 새로운 패키지가 필요했습니다.
4. 단 하나의 스크립트 모듈로써, 기능 추가 수정에 보수적이었습니다.

#### 어떤 점이 바뀌었나요?

1. 각 기능들을 모듈화하여, 필요한 기능만 사용할 수 있게 되었습니다.
2. 사용환경이 제한적이지 않습니다. react native, react, next 등 어떤 환경에서도 사용할 수 있습니다.
3. 기능에 대한 추가 수정에 자유롭습니다. 새로운 기능을 추가하기위해서, 모두가 쓸지말지, 사용할지말지에 대한 고민을 하지 않아도 됩니다. 그저 새로운 플러그인을 만들면 됩니다.

#### 어떠한 기능이 있나요?

1. cli - tokript 명령어로 각 플러그인으로 등록된 기능들을 사용할 수 있습니다.
2. api - 새로운 플러그인을 만들기 위한 api 가 제공됩니다. cli 로 실행되는 스크립트에 포함되는 설명, configuration, 명령어 옵션과 같은 설정들을 간편하게 정의할 수 있습니다.
