# gen:route (next page)

<DocIcons 
   github="https://github.com/TOKTOKHAN-DEV/toktokhan-dev/tree/main/packages/cli-plugins/gen-route-pages" 
   npm="https://www.npmjs.com/package/@toktokhan-dev/cli-plugin-gen-route-pages" 
   doc="/docs/cli-plugin-gen-route-pages"
   />


nextjs 프레임워크의 page 라우터 버젼에서 사용가능합니다.<br/>
pages 폴더를 조회하여 route 경로를 포함한 객체를 생성합니다.<br/>
개발자의 타이핑으로 경로를 가져오는것이 아니라 객체의 key 값으로 가져옴으로써 경로를 가져오는데 편의성과 안정성을 제공합니다.

```
pages
├── index.tsx
├── auth
│   └── index.tsx
├── login
│   └── index.tsx
└── social
    └── callback.tsx
```

위의 폴더 구조의 이미지는 아래의 객체로 변환됩니다.

```ts
export const ROUTES = {
  AUTH_MAIN: "/auth",
  MAIN: "/",
  LOGIN_MAIN: "/login",
  SOCIAL_CALLBACK: "/social/callback",
};
```
