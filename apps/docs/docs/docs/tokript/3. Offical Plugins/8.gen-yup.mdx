# gen:yup (RHF + Yup)


<DocIcons

대화형 cli 를 통해 일관된 형식의 yup 스키마를 생성하는 스크립트입니다.<br/>
변수 목적에 따라 정규식과 에러 텍스트를 연결하여 편의성과 안정성을 제공합니다.

```
✔ hook 이름 · use___
✔ 필요한 Schema (Space 선택 / Enter 확정) · 아이디 / 비밀번호
✔ 필수 Schema (Space 선택 / Enter 확정) · 아이디

=> src/generated/hooks/use___.ts
```
