# Introduction

[**Toktoken Figma Community**](https://www.figma.com/community/plugin/1369490322908230515/toktoken-figma-design-token-extractor?searchSessionId=lvyx7kh5-xhajkiwlic)

<img src={TokenThumbnail} alt="token-thembnail" style={{ width: 600 }} />

Toktoken은 Figma에서 정의된 디자인 시스템을 JSON 형식으로 파싱해주는 플러그인입니다.

Toktoken을 사용하면 쉽게 색상과 타이포그래피에 대한 토큰을 추출하여 구조화된 JSON 형식으로 내보낼 수 있습니다. `Toktoken` 플러그인을 통해 디자이너와 개발자가 디자인과 개발 작업 간의 일관성을 유지할 수 있도록 도와줍니다.

Figma 프로젝트에서 원하는 토큰을 선택해 파싱된 JSON 파일을 로컬에 저장하거나 작업중인 GitHub 프로젝트 저장소에 업로드할 수도 있습니다.

향후 추가적인 디자인 토큰 유형을 확장할 계획이므로, 앞으로의 업데이트를 기대해주세요.
